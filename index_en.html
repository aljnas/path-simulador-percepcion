🌈 PATH Perception Simulator

This project turns a Raspberry Pi into an emotional simulator using light, sound, texture and scent. It uses an LED strip and remote interface to recreate symbolic scenes that transform your space.

🏖️ Beach

Warm oscillating light, wave sounds, sea breeze scent, soft towel. An emotional refuge triggered by light.

🚁 Helicopter

Blinking blue light, rotor hum, symbolic vibration. Ideal for escape or control sensations.

🕯 Ritual

Dim warm light, soft instrumental music, lavender scent, memory-bound object. A deep connection trigger.
