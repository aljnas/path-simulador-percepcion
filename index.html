🌈 PATH Simulador de Percepción

Este proyecto convierte una Raspberry Pi en un simulador emocional basado en luz, sonido, textura y aroma. Usa una tira LED y una interfaz remota para recrear escenas simbólicas que transforman el espacio.

🏖️ Playa

Luz cálida oscilante, sonido de olas, aroma a brisa marina, toalla suave. Un refugio emocional activado con una luz.

🚁 Helicóptero

Luz azul parpadeante, zumbido de hélices, vibración simbólica. Ideal para simular escape o control.

🕯 Ritual

Luz tenue y cálida, música suave, esencia de lavanda, objeto con memoria. Activa conexión profunda con tu historia.
